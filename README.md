# 🏡 Global Airbnb Performance Dashboard

## 📌 Project Overview

This Power BI dashboard provides a comprehensive analysis of Airbnb listings across major global cities. The project focuses on understanding listing trends, pricing patterns, host distribution, market share, customer ratings, and review insights to support data-driven decision-making.

---

## 📊 Dataset Information

| Metric | Value |
|----------|----------|
| Total Records | 280,000+ |
| Total Columns | 18+ |
| Cities Analyzed | 10 |
| Hosts | 182,000+ |
| Property Types | 144 |
| Reviews | 5 Million+ |

### Dataset Fields Used
- Listing ID
- Host ID
- City
- Neighbourhood
- Property Type
- Room Type
- Price
- Review Scores Rating
- Review Scores Accuracy
- Review Scores Cleanliness
- Review Scores Communication
- Review Scores Location
- Review Scores Value
- Review Scores Check-in

---

## 🎯 Business Objectives

- Analyze Airbnb market share across cities.
- Identify high-performing cities based on ratings.
- Compare average pricing across room types.
- Track listing growth trends over time.
- Evaluate customer satisfaction using review scores.
- Understand host and property distribution.

## 🔍 Key Insights

✔ Paris has the highest number of Airbnb listings.

✔ Hotel Rooms have the highest average price among all room types.

✔ Mexico City achieved the highest average rating score.

✔ Airbnb listing growth peaked around 2015–2016 before declining.

✔ Superhost properties contribute significantly to listing performance.
